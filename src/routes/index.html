Twitter

Online

Para compartilhar informações no Twitter da forma mais ampla possível, é oferecido a empresas, desenvolvedores e usuários acesso programático a dados do Twitter através de APIs (interfaces de programação de aplicativo).

Os dados do Twitter têm um caráter único de compartilhamento em relação a outras mídias sociais porque refletem as informações que os usuários escolheram compartilhar publicamente.

Documentação

/twitter/:assunto

GET
